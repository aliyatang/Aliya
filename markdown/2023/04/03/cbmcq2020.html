Q38

  • Skill 5.A
  • 4.1 Daily Video 1, 2 IPv6 isn’t for users to bypass security protocols or send data peer-to-peer. Instead it is for internet protocol ADDRESSES, IPv4 has 32-bit address length and IPv6 has 128-bit address length. IPv6 is more advanced and has better features, such as being able to provide an infinite number of addresses, thus allowing for devices to be connected.
